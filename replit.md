# Xavisa - Sustainable Design

## Overview
Xavisa is a sustainable furniture e-commerce website that transforms discarded bottle caps into unique chairs and tables. The site is built with React, TypeScript, Vite, and Tailwind CSS with shadcn/ui components.

## Project Structure
- `src/` - React frontend source code
- `public/` - Static assets
- `index.html` - Entry HTML file

## Tech Stack
- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + shadcn/ui components
- **Routing**: React Router DOM v6
- **State Management**: TanStack React Query
- **Forms**: React Hook Form with Zod validation

## Running the Project
The app runs on port 5000 with `npm run dev`.

## Deployment
Configured for static deployment - builds to `dist/` folder.
